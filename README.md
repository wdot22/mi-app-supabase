# Mi App Supabase

## Descripción
Este proyecto es una aplicación básica desarrollada con Node.js que se conecta a una base de datos en Supabase. Permite consultar e insertar datos en una tabla llamada **estudiantes**.

---

## Funcionalidades
- Conexión a base de datos en Supabase
- Consulta de registros
- Inserción de nuevos estudiantes desde código

---

## Tecnologías utilizadas
- Node.js
- Supabase
- JavaScript
- Git y GitHub
